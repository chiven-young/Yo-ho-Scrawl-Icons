# 一函碎念
Yo-ho Scrawl Icons
一些随便画的玩梗图标。

## 图标位置：assets/

| 位置 | 尺寸 | 格式 | 说明 |
| --- | --- | --- | --- |
| icons/ | 192*192px | png | 一倍图，制作手机主题时默认图标大小 |
| icons@4x/ | 768*768px | png | 高清四倍图 |
| macicns/ | 1024*1024px | icns | macOS app图标 |
